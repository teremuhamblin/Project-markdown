# Code of Conduct

Ce projet adopte un comportement simple et respectueux.  
Toute contribution doit suivre les principes suivants :

## 1. Respect
- Respecter les autres contributeurs
- Éviter les propos offensants ou discriminatoires
- Favoriser un environnement positif

## 2. Communication
- Être clair, concis et constructif
- Expliquer les décisions techniques
- Accepter les retours et critiques

## 3. Collaboration
- Aider les nouveaux contributeurs
- Documenter les changements
- Suivre les règles définies dans `CONTRIBUTING.md`

## 4. Signalement
Pour tout comportement inapproprié :  
Ouvrir une issue ou contacter le mainteneur du projet.

Merci de contribuer dans un esprit professionnel et bienveillant.
