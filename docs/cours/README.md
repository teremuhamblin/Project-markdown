# Cours Markdown

Ce dossier regroupe des cours complets pour apprendre à écrire et éditer en Markdown, du niveau débutant au niveau avancé.

## Contenu
- `bases/` — fondamentaux du Markdown
- `intermediaire/` — mise en forme avancée
- `avance/` — usages professionnels
- `exercices/` — entraînement progressif

Chaque dossier contient un README.md et plusieurs leçons thématiques.
