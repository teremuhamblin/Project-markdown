# Modules

Ce dossier regroupe les modules essentiels du projet.  
Chaque module fournit un contenu structuré, clair et directement exploitable.

## Contenu
- **Workflow d’écriture** — processus complet pour créer et maintenir un document
- **Guide formateur** — ressources et méthodes pour enseigner le Markdown
- **Normes internes** — standards de qualité et de cohérence

## Objectif
Centraliser les règles, méthodes et bonnes pratiques afin d’assurer une documentation professionnelle, homogène et facile à maintenir.
