# Projets Pratiques Markdown

Ce dossier contient plusieurs projets complets pour mettre en pratique le Markdown dans des contextes réels : documentation, GitHub, workflows, CI/CD et templates professionnels.

## Contenu
- `projet-1-documentation.md` — Créer une documentation complète
- `projet-2-github.md` — Utiliser Markdown dans GitHub (issues, PR, README)
- `projet-3-workflow-ci.md` — Générer de la documentation via CI/CD
- `projet-4-template-pro.md` — Construire un template Markdown professionnel

Chaque projet inclut :
- un objectif
- un cahier des charges
- des étapes guidées
- un rendu final attendu
