# Projet Pratique 4 — Template Markdown Professionnel

## 🎯 Objectif
Créer un template Markdown réutilisable pour tout type de projet.

## 📌 Cahier des charges
Le template doit contenir :
- un README modèle
- un PLAN modèle
- un STYLEGUIDE
- un TEMPLATE.md
- un dossier `docs/` modèle
- un dossier `project/` modèle

## 🧩 Étapes
1. Créer un dossier `template/`
2. Ajouter les fichiers modèles
3. Ajouter des commentaires dans chaque fichier
4. Ajouter un index global
5. Ajouter un guide d’utilisation

## 📝 Rendu attendu
Un template complet, propre, réutilisable.
