# HOWTO

Guide rapide pour utiliser efficacement ce template Markdown.

## 1. Créer un nouveau fichier
1. Ouvrez le dossier correspondant (`docs/`, `project/`, `writing/`)
2. Dupliquez `TEMPLATE.md`
3. Renommez le fichier
4. Remplissez les sections

## 2. Ajouter un nouveau dossier
1. Créez un dossier avec un nom clair
2. Ajoutez un `README.md` pour expliquer son contenu
3. Placez-y vos fichiers

## 3. Mettre à jour la documentation
- Ajoutez vos modifications dans `CHANGELOG.md`
- Vérifiez les liens internes
- Respectez le `STYLEGUIDE.md`

## 4. Publier ou partager
- Commitez vos changements
- Poussez sur GitHub
- Utilisez la fonction **template** pour générer de nouveaux projets

## 5. Conseils
- Gardez une structure simple
- Documentez chaque dossier
- Utilisez des exemples dans `writing/exemples.md`
