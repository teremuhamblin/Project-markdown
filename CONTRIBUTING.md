# Contribuer au projet

Merci de votre intérêt pour ce projet Markdown.

## 🧩 Règles de contribution
- Garder les fichiers simples et courts
- Respecter le style défini dans `STYLEGUIDE.md`
- Utiliser des titres clairs
- Faire des phrases courtes

## 🛠️ Processus
1. Créer une branche
2. Modifier ou ajouter un fichier
3. Vérifier la cohérence
4. Ouvrir une Pull Request

## 📄 Format des commits
Utiliser un format simple :
- `add: description`
- `fix: description`
- `update: description`

## 📬 Contact
Ouvrir une issue si nécessaire.
