# Styleguide Markdown

## Titres
- Utiliser `#` pour les titres principaux
- Ne pas dépasser 3 niveaux

## Paragraphes
- Courts et directs
- Une idée par paragraphe

## Listes
- Préférer les listes courtes
- Utiliser `-` pour les listes simples

## Code
- Utiliser des blocs triple backticks
- Indiquer la langue si nécessaire

## Liens
- Toujours en Markdown `[texte](url)`
