# Markdown Project

Projet minimaliste conçu pour écrire, organiser et structurer du contenu en Markdown.  
Il fournit une base simple, claire et méthodique pour créer des documents propres et cohérents.

## 📁 Contenu

- `PLAN.md` — plan du projet
- `STYLEGUIDE.md` — règles d’écriture
- `EXAMPLE.md` — exemples de mise en forme

## 🎯 Objectifs

- Écrire proprement
- Structurer efficacement
- Maintenir une cohérence Markdown

## 🛠️ Utilisation

Ouvrir les fichiers et suivre les modèles fournis.
