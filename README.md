# Markdown Project "MAJ"

Template minimaliste conçu pour créer, organiser et maintenir une documentation claire, structurée et professionnelle.  
Ce projet fournit une base complète pour rédiger des documents, des guides, des workflows et des plans, tout en respectant des conventions cohérentes.

---

## 🎯 Objectifs 
- Offrir une structure simple et scalable
- Standardiser l’écriture et la documentation
- Faciliter la contribution et la maintenance
- Servir de base pour tout type de projet Markdown

---

## 📁 Structure du projet 
Le dépôt est organisé en plusieurs dossiers fonctionnels :

- `docs/` — documentation technique (architecture, setup, roadmap)
- `project/` — structure interne et workflows
- `writing/` — règles d’écriture et exemples
- `assets/` — images, modèles et ressources
- fichiers racine — documents essentiels (README, PLAN, STYLEGUIDE, TEMPLATE…)

Consultez **INDEX.md** pour une vue complète.

---

## 🚀 Comment utiliser ce template
1. Cliquez sur **Use this template**
2. Créez votre dépôt basé sur cette structure
3. Dupliquez `TEMPLATE.md` pour créer de nouveaux documents
4. Respectez les conventions du `STYLEGUIDE.md`
5. Mettez à jour `CHANGELOG.md` à chaque modification
